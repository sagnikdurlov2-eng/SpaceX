## 📌 Project Description

Space Registry is a decentralized application built on the **Stellar Soroban smart contract platform** that enables users to register and manage ownership of unique space assets such as satellites, orbital slots, and future space infrastructure.

In a world where space commercialization is rapidly growing, this project provides a **transparent, tamper-proof, and trustless registry system** to track ownership and transfers of space-related assets.

---

## ⚙️ What it does

The smart contract allows users to:

* 🛰️ Register new space assets with a unique ID
* 👤 Assign ownership to a Stellar address
* 🔄 Transfer ownership securely to another user
* 🔍 Retrieve asset details directly from the blockchain

All operations are **fully on-chain**, ensuring transparency and security without relying on centralized authorities.

---

## ✨ Features

* 🔐 **Secure Ownership Control**
  Uses Soroban’s authorization system to ensure only asset owners can perform sensitive actions

* 🛰️ **Unique Asset Registration**
  Prevents duplicate registrations using asset IDs

* 🔄 **Ownership Transfer System**
  Allows seamless and secure transfer of assets

* 📊 **On-Chain Transparency**
  All data is stored and verifiable on the Stellar blockchain

* ⚡ **Fast & Low-Cost Execution**
  Built on Stellar’s Soroban for efficient smart contract performance
